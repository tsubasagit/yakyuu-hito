OBS Studio で大学野球のライブ配信にテレビ中継風のスコアボードを重ねて表示するための設定手順です。
大学野球のオンライン配信を担当する方なら、このガイド通りに進めるだけで設定完了します。

操作デモ動画

OBS でのセットアップから実際の操作までの流れを動画で確認できます。

前提条件・推奨環境

  • OBS Studio ver31 以上を推奨（Windows・Mac で動作確認済み）
  • カスタムドック機能が必要です（v28 以降で搭載）。ver31 未満では正しく表示されない場合があります
  • Windows / macOS 対応 — 公式ダウンロード
  • ARM 版 Windows も v31.1 以降で対応

大学野球テロップ の構成

大学野球テロップ は2つの画面で構成されています。

画面 用途 OBS での表示方法
コントロール スコア入力・操作パネル カスタムドック
オーバーレイ 透過スコアボード表示 ブラウザソース
カスタムドックとブラウザソースは OBS 内部で同じブラウザエンジンを共有しているため、リアルタイム同期が可能です。Chrome 等の外部ブラウザとは同期できません。
1

カスタムドックを追加（コントロール画面）

  1. OBS メニューバー → 「ドック」 → 「カスタムブラウザドック」
  2. 以下を入力して 「適用」 をクリック
項目値
ドック名大学野球テロップ
URLhttps://hito-inc.jp/univ-bso/#/control

OBS 画面内にコントロールパネルが表示されます。

2

ブラウザソースを追加（オーバーレイ）

  1. 画面下部の 「ソース」 の 「+」 ボタンをクリック
  2. 「ブラウザ」 を選択
  3. 名前を入力（例: 大学野球テロップ overlay）して「OK」
  4. プロパティ画面で以下を設定
項目値
URLhttps://hito-inc.jp/univ-bso/#/overlay
幅1920
高さ1080
カスタムCSS中身を全て削除して空欄にする
URL に #（ハッシュ）が含まれています。/#/overlay の部分を省略しないでください。
カスタムCSS はデフォルトで body { background-color: rgba... } が入っています。これを削除しないと背景が正しく透過されません。
3

接続を確認

設定が正しければ、以下の状態になります。

  1. カスタムドック右上の接続ステータスが 「オーバーレイ接続中」（緑） に変わる
  2. プレビュー画面にスコアボード・打順カード等のオーバーレイが表示される
  3. カスタムドックで操作するとオーバーレイにリアルタイム反映される
4

画面に合わせる

ブラウザソースのサイズが画面と合わない場合:

  1. プレビュー画面のオーバーレイ（赤枠）を 右クリック
  2. 「変換」 → 「画面に合わせる」
5

パネルの位置・サイズを調整

OBS のブラウザソース内ではパネルをドラッグできないため、コントロールパネル上部の 「🎬 配信画面に出すパネルを選ぶ」 から、パネル毎に ON/OFF・サイズ・位置（X/Y）を数値で調整します。

  1. 「全体スケール」 スライダーで全パネルの拡大率をまとめて変更（基準は 1.0x）
  2. 各パネルカードの 「サイズ」 スライダーで個別の倍率を調整
  3. 各パネルカードの 「位置」 の X / Y 欄に数値を入力して配置（1920×1080 基準の座標）
  4. バッター・ピッチャーは打順表の 「打席」「登板」 ボタンで表示を切り替えます（カード自体に ON/OFF はありません）

全体スケール横の 「位置リセット」 ボタンで、全パネルの位置・サイズを既定値に戻せます。

トラブルシューティング

症状対処法
「オーバーレイ未接続」のまま ブラウザソースの URL に # が入っているか確認。カスタムドックを右クリック→再読み込み
背景が透過されない ブラウザソースのプロパティで「カスタムCSS」を空欄にする
オーバーレイがはみ出る オーバーレイを右クリック →「変換」→「画面に合わせる」
Chrome で操作しても OBS に反映されない Chrome と OBS は別ブラウザのため同期不可。必ず OBS カスタムドックから操作してください
パネルの位置を変えたい コントロールパネル上部の「🎬 配信画面に出すパネルを選ぶ」で、各パネルの X/Y・サイズを変更
