# yakyuu-hito

大学野球オンライン配信向けのOBSブラウザソース用オーバーレイツール。
株式会社ひと（LINE系スポーツナビ大学野球配信事業）専用カスタマイズ版です。

[tsubasagit/yakyuu](https://github.com/tsubasagit/yakyuu) v0.3.0 から分岐し、
学生オペレーター運用に合わせて機能を絞り込んだ派生プロジェクトです。

## 背景

大学野球は「プロ野球選手の6〜7割が大学生ドラフト候補」という重要な舞台でありながら、
配信予算は高校野球（年間14億円規模）の数千万円と限られており、
現場は大学生スタッフが撮影・配信を兼任する運用になっています。

本ツールはOBS上で動作し、以下の4要素を大学生でも迷わず操作できるUIで提供します。

## 必須4要素

1. **BSO（ボール・ストライク・アウト）**
2. **スコアボード**
3. **打者名**
4. **スターティングラインナップ一覧**

各要素は個別に表示/非表示切替が可能です。

## 画面

- `/#/control` — スコアキーパー操作パネル
- `/#/overlay` — OBS用オーバーレイ（透明背景）

## 技術スタック

| 技術 | 用途 |
|---|---|
| React | UI |
| TypeScript | 型安全 |
| Vite | ビルド |
| Tailwind CSS | スタイリング |
| Zustand | 状態管理 + localStorage 永続化 |
| BroadcastChannel API | タブ間リアルタイム同期 |

## 開発

```bash
npm install
npm run dev    # http://localhost:5173
npm run build  # 本番ビルド
```
